POST /api/decks/create (alias: /api/decks/create-single)

Create a single shuffled draw pile with empty discard and null current card.

deck_names
Required string[] — deck names from Postgres catalog.
shuffle_seed
Optional. null | finite number | non-empty string. Omit or null → random shuffle each recycle. Set only on create; round-trip on pile JSON thereafter.
Response
draw_pile[], discard_pile[], current_card (object|null), shuffle_seed 
(response here)
POST /api/decks/create-dual

Create exploration and item draw piles with rich card fields when present in DB.

deck_names
Required string[].
shuffle_seed
Same semantics as single create.
Response
exploration_drawpile, exploration_discardpile, exploration_current_cards, item_drawpile, item_discard_pile, item_current_cards, shuffle_seed 
(response here)
POST /api/single/draw

Draw one card; updates current_card, draw_pile, discard_pile.

Body
Full single pile JSON from create (round-trip entire object).
Behavior
One card per request; shuffle_seed carried on pile for recycle RNG.
(response here)

mill_count
